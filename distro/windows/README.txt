# otgui module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otgui\examples\otgui

Installation
  Silent install can be done with the switch /S (e.g. openturns-otgui-0.0-py27.exe /S).
  To test otgui module installation, in an OpenTURNS python console type:
    import otgui
    print otgui.__version__

Uninstallation
  OTGui module will be automatically uninstalled at OpenTURNS removal.
  OTGui module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
