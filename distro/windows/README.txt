# persalys module

Examples
  Template examples can be found here: C:\Python37\Lib\site-packages\persalys\examples

Installation
  Silent install can be done with the switch /S (e.g. persalys-0.0-py37-x86_64.exe /S).
  To test persalys module installation, in an OpenTURNS python console type:
    import persalys
    print persalys.__version__

Uninstallation
  Persalys module will be automatically uninstalled at OpenTURNS removal.
  Persalys module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
